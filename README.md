# countdown-numbers-solver
Solver for the Numbers round on Countdown
 
 
#### Input Format:  
Number of values  
{ Array of N values }  
Target value  

#### Output:  
Mathematical expression to achieve target value (if achievable)

### Example Usage  
#### Input:
4   
5 10 25 100  
123  

#### Output:
((25 - (10 / 5)) + 100)
